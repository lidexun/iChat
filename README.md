### ichat(一对一聊天/前后端分离)

> 写在最前！你可以看到用 Vue3 写一个聊天功能，持续更新到完整聊天室的功能~持续更新中

### 技术栈

前端：`Vue3`+`Vant`+`Vite`

后端：`Nodejs(express)`+`mongoDB`

通信：`Websocket(socket.io)` + `axios`
